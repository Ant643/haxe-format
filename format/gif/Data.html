typedef Data

package format.gif

Gif data.

version:Version

Gif version. There is only 2 Gif version exists. 87a and 89a. 87a have less features and does not support any extensions. Unknown version is adviced to be interpreted as newest (89a) official version.

logicalScreenDescriptor:LogicalScreenDescriptor

Information about logical screen of Gif that provides basic information about Gif.

@:optionaloptionalglobalColorTable:Null<ColorTable>

Global color table used for Gif. Present only if Logical Screen Descriptor contained global color table flag. Note that this color table not always present since frames can contain local color tables that overrides global color table.

blocks:List<Block>

List of Gif data blocks.
